# Source Map

Last reviewed: 2026-04-26

## Verify Current Docs First

Before giving version-sensitive advice, open the relevant official page and check the current WordPress, Gutenberg, WP-CLI, Plugin Check, Composer, npm package, PHPUnit, or CI guidance. This is required for APIs introduced or changed in recent WordPress releases, especially Block Editor metadata, Interactivity API, script modules, Plugin Check rules, and WP-CLI scaffolding output.

## Source Use Policy

- Summarize and link. Do not copy official docs into generated output.
- Prefer official docs over blog posts. Use community posts only to clarify examples, never as the source of truth.
- When a package command or option matters, verify the current package page before recommending exact flags.
